# Map school year to ALSDE report year

ALSDE uses end year (e.g., 2024 for 2023-24 school year)

## Usage

``` r
get_report_year(end_year)
```

## Arguments

- end_year:

  End year of school year (e.g., 2024 for 2023-24)

## Value

Same year (passthrough for consistency with other state packages)
