# Download raw assessment data from ALSDE

Downloads school-level ACAP Reading assessment data from the Alabama
State Department of Education Alabama Achieves website.

## Usage

``` r
get_raw_assess(end_year)
```

## Arguments

- end_year:

  School year end (2023-24 = 2024). Valid values are 2022-2025.

## Value

Data frame with raw assessment data
